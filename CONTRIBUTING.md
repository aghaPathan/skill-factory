# Contributing to Skill Factory

## Adding a New Skill

### 1. Create the skill directory

```bash
mkdir -p skills/<your-skill-name>/evals
```

### 2. Write SKILL.md

Create `skills/<your-skill-name>/SKILL.md`:

```yaml
---
name: your-skill-name
description: When to trigger this skill — be specific about activation criteria
version: 1.0.0
tags: [relevant, tags]
platforms: [claude-code, gemini-cli, codex-cli]
author: your-github-username
---

# Your Skill Name

Instructions for the AI agent...
```

### 3. Add evaluation test cases

Create `skills/<your-skill-name>/evals/evals.json`:

```json
{
  "skill_name": "your-skill-name",
  "evals": [
    {
      "id": 1,
      "prompt": "Test prompt that should trigger this skill",
      "expected_output": "What success looks like",
      "files": [],
      "expectations": [
        "Specific assertion about behavior",
        "Another assertion"
      ]
    }
  ]
}
```

Aim for 3+ test cases covering primary use cases.

### 4. Build and validate

```bash
npm install          # First time only
npm run validate     # Check frontmatter
npm run build        # Generate dist/ files + update README catalog
```

### 5. Submit a PR

- Branch from `main`
- Include `skills/` source AND `dist/` generated files
- PR description should explain what the skill does and when it triggers

## PR Checklist

- [ ] `skills/<name>/SKILL.md` has required frontmatter (`name`, `description`)
- [ ] `skills/<name>/evals/evals.json` has 3+ test cases
- [ ] `npm run validate` passes
- [ ] `npm run build` succeeds and dist/ is committed
- [ ] README catalog table is updated (auto-generated by build)

## Modifying Build Tooling

The build pipeline lives in `src/`. After changes:

```bash
npm run build        # Verify build still works
```

### Adding a Platform Adapter

1. Create `src/platforms/<platform-id>.ts` implementing the `PlatformAdapter` interface from `src/platforms/base.ts`
2. Add to the `PLATFORMS` array in `src/platforms/index.ts`
3. Run `npm run build` to verify
4. Update README.md install instructions for the new platform
